Chapter 3: Images

Still images are generated by the computer in two ways: Bitmaps (raster) or Vector graphics. Understanding the difference is key to choosing the right format for a project.

1. Bitmaps (Raster)

A Bitmap is a data matrix describing individual dots (pixels) that are the smallest elements of resolution on a computer screen.

  • Ideal for photo-realistic images and complex color gradients.
  • Quality depends on resolution; they lose clarity when resized (pixelation).
  • Dithering: A process where the color value of a pixel is changed to the closest matching color in a target palette using a mathematical algorithm.

2. Vector Graphics

Vector-Drawn Graphics are created from geometric objects (lines, rectangles, etc.) using mathematical formulas.

  • Can be resized infinitely without any loss of quality.
  • Requires significantly less storage space than high-resolution bitmaps.
  • Best for logos, icons, and illustrations.

Color Depth & Models

Bit Depth refers to the number of bits used to describe a color; more bits provide higher color depth and photo-realism.

Color Models

  • Additive Color (RGB): Created by combining red, green, and blue light. This is the standard for monitors and television screens.
  • Subtractive Color (CMYK): Created by combining cyan, magenta, yellow, and black ink. This is the standard used for printing.

3-D Modeling & Animation

Creating 3D imagery involves a specific pipeline of technical processes:

  1. Modeling: The initial stage of drawing a shape and setting camera views or lighting.
  2. Extruding: Extending a 2D shape along a defined path to create a 3D object.
  3. Lathing: Rotating a profile of a shape around a defined axis to create a 3D object (like a vase or a bowl).
  4. Rendering: The final, compute-intensive process of producing the finished image or scene from the 3D model.
Morphing: A specialized technique used to manipulate still images to create smooth, animated transformations from one image to another.

The Clipboard

A temporary memory area used by the operating system to store copy, cut, and paste data, allowing images to be moved between different authoring tools.
