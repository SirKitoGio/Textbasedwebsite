Chapter 1: What is Multimedia?

In the context of this review, Multimedia is defined as any combination of text, art, sound, animation, and video delivered to you by computer or other electronic or digitally manipulated means.

Key Concept: When you weave together these elements, you can "electrify the thought processes of peopleâ€™s minds" by engaging multiple senses simultaneously.

Types of Multimedia

  • Interactive Multimedia: A system where the user is given the option of controlling what and when the elements are delivered.
  • Hypermedia: A combination of hypertext, graphics, audio, video, and interactivity resulting in a non-linear experience.
  • Linear Project: A project that is not interactive; it starts at the beginning and runs to the end (e.g., a cinema movie).
  • Non-Linear Project: An interactive project where users have navigational control over the content (e.g., games, courseware).

The Five Core Elements

Multimedia projects are built using five fundamental building blocks:

  • Text: The most basic element used for content, menus, and navigation.
  • Graphic: Two-dimensional figures or illustrations that can be produced manually or via computer.
  • Audio: Produced by vibration and perceived by hearing; includes speech, sound effects, and music scores.
  • Animation: The illusion of motion created by the consecutive display of static images.
  • Video: The technology of capturing and reconstructing moving pictures, typically involving photo-realistic live recordings.

Multimedia Development & Technology

Authoring Tools

To bring these elements together, developers use Authoring Tools. This software is used to merge individual multimedia elements into a cohesive project and manage user interaction.

Virtual Reality (VR)

Virtual Reality is an extension of multimedia that uses goggles or helmets to place the user "inside" a lifelike experience, creating an immersive 3D environment.

Common Multimedia Products

Multimedia is often packaged into specific formats based on their use case:

  • Briefing Products: Small, straightforward, linear products used to present information quickly and concisely.
  • Kiosk Products: Products stationed in public places that allow users to find information or perform transactions interactively.

Multimedia Delivery

Multimedia projects can be delivered in two primary ways:

  1. Offline: Via storage media like CD-ROM or DVD.
  2. Online: Over the World Wide Web (using HTML!).
