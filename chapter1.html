Chapter 1: What is Multimedia?

In the context of this review, Multimedia is defined as a combination of five basic elements: text, graphic art, sound, animation, and video. These elements are delivered to you by computer or other electronic means.

Key Concept: When you weave together these elements, you can "electrify the thought processes of peopleâ€™s minds".

Types of Multimedia

  • Interactive Multimedia: This allows the end-user (you) to control what and when the elements are delivered.
  • Hypermedia: When a project includes a structure of linked elements through which the user can navigate, it becomes hypermedia.
  • Linear Multimedia: Projects that are not interactive and proceed from beginning to end, like a cinema movie.
  • Non-Linear Multimedia: Projects that allow users to jump around and follow their own path.

The Five Core Elements

  • Text: The written content of multimedia projects.
  • Graphic Art: Visual elements like images and illustrations.
  • Sound: Audio elements such as music and speech.
  • Animation: Moving images or graphics that create motion.
  • Video: Recorded moving images with sound.

Multimedia Delivery

Multimedia projects can be delivered in two primary ways:

  1. Offline: Via storage media like CD-ROM or DVD.
  2. Online: Over the World Wide Web (using HTML!).
