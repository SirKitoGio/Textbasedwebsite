Chapter 2: Text

Text is the simplest but most important element for communication in multimedia. It provides the core meaning and navigation, ensuring users aren't left guessing the purpose of images or sounds.

Typefaces VS Fonts

In digital design, these two terms are specific:

  • Typeface: A family of graphic characters that usually includes many sizes and styles (e.g., Bookman Old Style).
  • Font: A collection of characters of a single size and style belonging to a typeface family (e.g., Arial 18 point Bold).

Font Terminology

Understanding how characters are measured and positioned is crucial for layout design:

  • Baseline: The imaginary line on which the bases of characters are arranged.
  • Leading: The vertical distance between successive baselines (line spacing).
  • x-height: The distance between the baseline and the top of a lower-case letter 'x'.
  • Ascenders/Descenders: Strokes that rise above the x-height or drop below the baseline.

Text Manipulation & Spacing

Adjusting the space between characters improves readability and aesthetics:

  • Kerning: The adjustment of space between specific pairs of letters (e.g., 'AV').
  • Tracking: The overall spacing between all characters in a block of text.

Digital Text Concepts

Anti-aliasing: A technique that blends font edges into the background color to minimize "jagged" edges (jaggies) and make text look smoother.
  • OCR (Optical Character Recognition): A system designed to translate images of handwritten or typewritten text into machine-editable text.
  • Font Mapping: Specifying a substitution font if the original font is missing on a user's computer.

Character Encoding Standards

To display text, computers use binary codes to represent characters:

  1. ASCII: A 7-bit binary code representing 128 characters. It is the basic American standard.
  2. Unicode: A 16-bit universal standard that supports over 96,000 characters, including multi-language characters and symbols.

Hypertext

Formatted text that enables a related set of documents (pages) to be linked together via hotspots or hyperlinks, creating a non-linear navigation experience.
