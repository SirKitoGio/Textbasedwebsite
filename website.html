Multimedia and HTML: Overview

Multimedia is a combination of text, graphic art, sound, animation, and video delivered by computer or other electronic means. It is categorized into two types: Linear (not interactive, like a movie) and Non-linear (interactive, like a game).

Categories of Multimedia:

Linear: Not interactive, this is where users can just watch and everything else happens in a predetermined sequence.

Non-linear: Interactive, this is where users can navigate and interact with the content in various ways.

Web Development (HTML)

To deliver multimedia on the web, we use HTML (HyperText Markup Language). It uses "tags" to structure content so that browsers can display text and media correctly.
