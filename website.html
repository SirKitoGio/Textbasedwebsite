Multimedia and HTML: Overview

Welcome to the ReviewPedia. This wiki serves as a comprehensive study guide for the fundamentals of multimedia systems and web development. Multimedia is defined as any combination of text, art, sound, animation, and video delivered by computer or electronic means.

The Ted Nelson Concept: The term Hypertext was coined by Ted Nelson around 1965, referring to text that links to other pieces of information, allowing for a non-linear reading experience.

Key Definitions

Linear vs. Non-linear

  • Linear: Projects that are not interactive and proceed from beginning to end, like a cinema movie or a Briefing Product.
  • Non-linear: Interactive projects where the user has navigational control, such as games, Kiosk Products, and courseware.

Web Delivery

In modern computing, the primary method for delivering multimedia online is via HTML (HyperText Markup Language). By using specific tags, we can weave together the core elements to create an "electrified" user experience.
