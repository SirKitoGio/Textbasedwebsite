HTML Basics

HTML stands for HyperText Markup Language. It is a text markup language (not a programming language) used to tell web browsers how to display and structure a page.

1. Tags vs. Elements

It is important to distinguish between these two fundamental concepts:

  • HTML Tags: Commands surrounded by angle brackets (< and >) that specify how a document should be formatted.
  • HTML Elements: The combination of the start tag, the content, and the end tag (e.g., <b>Bold Text</b>).

2. Core Document Structure

Every HTML5 document requires four core structural tags to function correctly:

  • <html>: The container for the entire document; tells the browser where the HTML starts and ends.
  • <head>: Contains "metadata" or header information that is not displayed directly in the browser window (like links to CSS).
  • <title>: Defines the text displayed in the browser's title bar or tab.
  • <body>: Contains all the visible text, images, and elements of the web page.

3. Content & Formatting Tags

These tags are used inside the <body> to organize information:

  • Headings: <h1> through <h6> define the hierarchy of your content.
  • Paragraphs: <p> is used for standard blocks of text.
  • Bold: <b> is used to display text in a bold font.
  • Lists: Use <ul> for unordered (bullets) and <ol> for ordered (numbers) lists. Each item is wrapped in an <li> tag.

4. Attributes: Links and Images

Attributes provide additional information about an element and are always found in the start tag.

Hypertext Links

Links are created with the <a> tag. The href attribute specifies the destination URL or file.

<a href="chapter1.html">Go to Chapter 1</a>

Images

The <img> tag is used to embed visual media. It requires the src (source path) and alt (alternative text) attributes.

<img src="photo.jpg" alt="Description of image" width="500">
