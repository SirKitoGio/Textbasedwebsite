HTML Basics

HTML stands for HyperText Markup Language. It is the backbone of any website, providing the structure for content.

1. Document Structure

Every HTML5 document should follow this basic skeleton:

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <!-- Content goes here --> </body> </html>

2. Common Tags

  • <h1> to <h6>: Defines headings, with H1 being the most important.
  • <p>: Defines a paragraph of text.
  • <a>: Creates a hyperlink using the href attribute.
  • <img>: Embeds an image using the src and alt attributes.
  • <ul> and <ol>: Creates unordered (bulleted) and ordered (numbered) lists.

3. Working with Links

Links are what make the "Hypertext" in HTML. To link to another page in your folder, use:

<a href="chapter1.html">Go to Chapter 1</a>
